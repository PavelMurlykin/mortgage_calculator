# Mortgage Calculator (Ипотечный калькулятор)

Современное и удобное Android-приложение для расчета параметров ипотечного кредита. Позволяет быстро оценить ежемесячный платеж, максимальную стоимость недвижимости и просмотреть детальный график платежей.

## Основные возможности

- **Гибкий расчет**: 
  - Расчет ежемесячного платежа на основе стоимости объекта.
  - Расчет максимально доступной стоимости недвижимости на основе желаемого платежа.
- **Два типа платежей**: Поддержка аннуитетных и дифференцированных платежей.
- **Первоначальный взнос**: Возможность задания взноса как в рублях, так и в процентах (с функцией "заморозки" процента).
- **Подробный график**: Просмотр детального графика платежей с разбивкой по месяцам (основной долг, проценты и остаток).
- **История расчетов**: Сохранение и удаление выполненных расчетов для быстрого доступа к ним в будущем.
- **Настройки под себя**: Настройка шагов изменения суммы, процентов и типа платежа по умолчанию.

## Технологический стек

Проект написан с использованием современных инструментов разработки под Android:

- **Kotlin** — основной язык разработки.
- **Jetpack Compose** — современный декларативный UI.
- **Material 3** — дизайн-система приложения.
- **Navigation Compose** — навигация между экранами.
- **Room** — локальная база данных для хранения сохраненных расчетов.
- **DataStore** — хранение настроек пользователя.
- **Kotlin Coroutines & Flow** — реактивное обновление UI и асинхронная работа.
- **ViewModel** — архитектурный компонент для управления состоянием UI.

## Скриншоты и установка

Приложение поддерживает современные версии Android (API 24+). Готовый APK файл для установки можно найти в разделе [Releases](https://github.com/PavelMurlykin/mortgage_calculator/releases).

## Как собрать проект самостоятельно

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/PavelMurlykin/mortgage_calculator.git
   ```
2. Откройте проект в **Android Studio**.
3. Создайте свой ключ подписи или используйте отладочную сборку (debug).
4. Запустите проект на эмуляторе или реальном устройстве.
